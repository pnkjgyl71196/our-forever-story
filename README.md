# Our Forever Story ❤️

A Disney/Pixar-style interactive proposal website.

## Features
- Cute panda and teddy bear illustrations
- Animated hearts
- Interactive "Reasons I Love You"
- Romantic proposal section
- Save response locally
- Download response as JSON
- Mobile responsive
- 100% free to host on GitHub Pages

## Customize
Open `index.html` and replace:
- "For The Love Of My Life"
- Timeline text
- Letter text
- Your names and memories

You can also add your own photos inside the `assets/` folder and reference them in `index.html`.

## Run locally
Double-click `index.html`.

## Host on GitHub Pages

1. Create a GitHub account.
2. Create a new repository (for example: `our-forever-story`).
3. Upload all files from this folder.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/ (root)**
6. Click **Save**.
7. Wait 1–2 minutes.
8. Your website will be live at:
   `https://YOUR-USERNAME.github.io/our-forever-story/`
